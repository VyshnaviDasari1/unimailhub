About UniMailHub

UniMailHub is a web-based email management system inspired by Gmail. It allows users to send and receive emails with attachments, manage inboxes, starred mails, search emails, and securely delete messages.

Built using Spring Boot, Thymeleaf, and MySQL, this application demonstrates a real-world enterprise email workflow.
